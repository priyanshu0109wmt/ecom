AgriCart – Full Stack E-Commerce Platform
AgriCart is a full-stack e-commerce web application designed for selling agricultural products like fruits,
vegetables, and dairy items. It provides a seamless experience for users to browse products, manage carts, and place orders with a clean and modern UI.

Features
🔐 User Authentication (Signup & Login)
🛍️ Product Listing with Categories
🛒 Add to Cart with Quantity Management
➕ Increase / Decrease Cart Items
💳 Checkout System
📦 Order History Tracking
🎨 Modern Responsive UI (recently upgraded)
⚡ Real-time frontend-backend integration

Tech Stack
Frontend
HTML5
CSS3 (Modern UI with Flexbox & Grid)
JavaScript (Vanilla JS)
Backend
Node.js
Express.js
Database
MySQL
Deployment
Frontend: Netlify
Backend: Railway
